## Installation

```sh
npm install --save bs-dgraph-js-http
```

Then add `bs-dgraph-js-http` to `bs-dependencies` in your `bsconfig.json`:
```js
{
  ...
  "bs-dependencies": ["bs-dgraph-js-http"]
}
```
